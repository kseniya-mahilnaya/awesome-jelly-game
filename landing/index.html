About

Awesome Jelly is a classic one-screen platformer game built on game framework Phaser.js which uses both Canvas and WebGL renderer. Technologies used in the game:

  • JavaScript (ES6)
  • Phaser.js
  • EZGUI.js
  • WebGL / Canvas
  • HTML5 + CSS3

Menu screen

Main menu screen. Play button - start the game, Controls button - show game controls.

Level 1

This is the first level of the game which is called "Under the sea." Collect as many stars as possible killing the enemies at the same time. And the main aim is to put the key into the lock to go to the next level and to stay alive. You have only 3 lives. Be careful some platforms are movable on this level.

Level 2

This is the second level of the game which is called "Jelly's breakfast." Collect as many stars as possible killing the enemies at the same time. And the main aim is to put the key into the lock to go to the next level. And to stay alive of course! Keep it fighting!

Level 3

This is the third level of the game which is called "Night of adventures." Collect as many stars as possible killing the enemies at the same time. And the main aim is to put the key into the lock to win the game. Be careful some platforms are movable on this level. Stay up!

Controls screen

Use this keys to move your Jelly! To kill the enemy jump up.

Game over screen

This is an example of game over screen. Game over state is initialized when the jelly has spent all three lives. All three stars are empty, so no level passed.
